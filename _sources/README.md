# DBSII_Script

View current script on [GitHub Pages](https://luh-dbs.github.io/DBSII_Script/intro.html).

## Quickstart 
Install Jupyter Book:
```
pip install -U jupyter-book
```
Build book:
```
jupyter-book build DBSII_Script
```
Add to GitHub Pages:
```
ghp-import -n -p -f _build/html
```
